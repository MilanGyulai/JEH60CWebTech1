Git parancsok

  • git clone (a github repository elérése)
  • git pull (a git repository aktuális branch-én lévő frissítés)
  • git status (kiírja az állapotát a directory-nak és a stageing-nek)
  • git add . (mindent hozzáad az aktuális mappában (felállítja a stage-re))
  • git commit -m "" (git commit ezzel commit-toljuk a változtatásainkat és commit üzenetet adunk hozzá)
  • git push (fetch-eli az összes branchet a remote-ról)
  • git branch (kiírja a brancheket a repository-n)
  • git checkout (másik branch-re tudunk vele váltani)
  • git log (Megmutatja a commit-okat a branch életében)
  • git rm [file név] (törlése a file-nak a projektből)
  • git stash (El tudjuk menteni az aktuális állapotot azon a branch-en amin éppen vagyunk)
  • git stash pop (A stash-ből ki tudjuk venni a változtatásainkat(az utolsó változtatást))
  • git stach discar (A stash-ből ki tudjuk törölni a stack tetejéről)
